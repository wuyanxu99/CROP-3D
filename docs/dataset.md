# Dataset

The Git repository includes only compact demo assets.

The full dataset should be distributed through a dedicated archive such as Zenodo, institutional storage, Hugging Face Datasets, or GitHub Releases, depending on review and sharing permissions.

Recommended metadata to add before public release:

- Dataset name and version.
- Capture date range and scene description.
- Camera setup and calibration notes.
- File organization.
- Download link or request contact.
- License or data-use terms.

The raw `data/` directory from the research workspace is intentionally not copied here.
